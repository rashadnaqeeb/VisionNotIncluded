# Game Mechanics Reference

This directory contains three sources of game knowledge. Use all relevant sources when answering questions.

## Topic files (65 .md files)

Exact values and formulas sourced from decompiled code. Authoritative for numerical data (thermal conductivity, mass thresholds, internal constants). Check here first for precise values.

## wiki/ (1,442 .md files)

Articles from the ONI wiki.gg wiki (CC BY-SA 3.0), one file per article. Broader coverage: building descriptions, strategies, critter behaviors, achievements, recipes, tips. Use for general game knowledge, obscure topics not in the topic files, and player-facing descriptions. Searchable with grep.

## Jahws' guides (4 jahws-*.md files)

Verbatim text of Jahws' Steam strategy guides: early game, mid game, dupe builds, and construction patterns. These contain inline screenshot URLs. When answering a player question that leads to a guide section, load and describe nearby screenshots with the Read tool -- they show base layouts, overlay views, automation wiring, etc. that the text alone doesn't fully convey.
